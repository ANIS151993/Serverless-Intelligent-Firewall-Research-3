\begin{abstract}
Serverless environments demand adaptive security controls that can process short-lived workloads, identity context, and multi-cloud policy state at runtime. This paper presents an autonomous self-learning serverless intelligent firewall that combines REST API-driven open-source threat intelligence, a multi-paradigm learning engine, and federated zero-trust policy propagation. The system provides explainable risk scoring, cloud-specific mitigation orchestration, and feedback-based weight adaptation to improve resilience under evolving attacks.
\end{abstract}

\begin{IEEEkeywords}
serverless security, intelligent firewall, threat intelligence, zero trust, federated policy, adaptive machine learning
\end{IEEEkeywords}

\section{Introduction}
The proposed architecture integrates threat telemetry, hybrid model scoring, and federated policy enforcement to provide real-time security decisions across distributed cloud surfaces.

\section{Conclusion}
The research artifact demonstrates a practical path for reproducible serverless security with autonomous adaptation and policy coherence.
